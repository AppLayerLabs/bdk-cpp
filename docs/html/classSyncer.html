Detailed Description

Helper class that syncs the node with the network.

This is where the magic happens between the nodes on the network, as the class is responsible for syncing both, broadcasting transactions and also creating new blocks if the node is a Validator. Currently it's single threaded, meaning that it doesn't require mutexes.

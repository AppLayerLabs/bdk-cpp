Detailed Description

template<unsigned N>
class FixedStr< N >

Abstraction of a fixed-size string (FixedStr<10> would have exactly 10 characters, no more, no less).

This class is used as a base for both classes inheriting it (e.g. Hash, Signature, etc.) and aliases (e.g. PrivKey, PubKey, etc.).

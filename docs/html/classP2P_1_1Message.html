std::string P2P::Message::_rawMessage
private

The internal message data to be read/written, stored as bytes (1 byte = 2 chars).

The structure of a message is as follows:

0x 00 0000000000000000 0000 00000000000000000000000000000000... What: Request Flag Random ID Cmd ID Data... Chars: 2 16 4 X Bytes: 1 8 2 X/2

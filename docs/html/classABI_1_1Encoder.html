ABI::Encoder::Encoder ( ABI::Encoder::EncVar  data,
std::string  func = "" 
)

Constructor.

Automatically encodes the data during construction. Throws on error.

Parameters
dataA list of variables to encode.
func(optional) The full function header to encode. Defaults to an empty string.

Handle each data type and value, like this:

  • Split parsing in two: static vars + offsets (appended to this->data), and dynamic vars (appended to dynamicStr)
  • Consider each 32-byte/64-char space, thus each declared item in the array, as a "line" (for easier understanding)
  • Count a "global" dynamic offset (nextOffset), starting from (32 bytes * starting number of lines), and adding every (32 bytes * number of lines from current item) for each parsed item

◆ encodeBytes()

std::string ABI::Encoder::encodeBytes ( std::string  bytes) const
private

Encode a raw bytes or UTF-8 string into Solidity ABI format.

Solidity bytes and string types are parsed the exact same way, except string REQUIRES an utf8ToHex() conversion. This is done internally using isHex(strict) to discern from both types. If bytes, APPEND 0x to the start of the string. If string, REMOVE 0x from the start of the string.

Parameters
bytesThe raw bytes or UTF-8 string to encode.
Returns
The encoded hex bytes or string, padded to the nearest multiple of 32 bytes to the RIGHT.

std::string ABI::Encoder::encodeFunction ( std::string  func) const
private

Encode a function header into Solidity ABI format.

Requires the full function header, no spaces between args (e.g. func(arg1,arg2)). The function will SHA3-hash the header and return the first 4 hex bytes of the hash (aka the "functor").
